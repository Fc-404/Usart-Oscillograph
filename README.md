# 一种串口示波器
> 可以通过USART接收模拟电平信号，然后显示出来
> 
> 按左右键可以调时间显示精度，1 - 10 倍周期
> 
> 按空格键可以切换显示模式，有散点式和折线式
>
> USART发送的数据应当是`±5V`以内的浮点数（发送时是字符型），精度一般在`0.001`
>
> 时间显示精度和USART发送频率有关，每一块的时间间隔是`发送周期 * 时间显示精度 * 100`

## 自定义
> 代码默认串口是COM3，切换串口需要修改源码重新编译
``` gcc
gcc ./main.c -lgdi32 -lpthread 
````
> 参数中至少包含两个库，可以加入其他优化参数，如`-Ofast`
